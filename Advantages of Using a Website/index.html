Advantages of using a Network

There are numerous advantages of using a network. I will outline some of these below.

  • Peripeheral devices can be shared which saves money.
  • Software licenses are likely to be cheaper than buying standalone ones.
  • Files can easily be shared between users.
  • Network users can communicate by email.
  • Data can be easily backed up onto the file server.
